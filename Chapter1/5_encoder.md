## 기본개념

| 항목                       | 내용                                                                                                                                                                          | 핵심 포인트                                                                                                            |
| ------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------- |
| 엔코더 모터의 정의               | 엔코더 모터는 일반적으로 모터에 엔코더가 부착된 형태이며, 엔코더를 통해 회전량, 속도, 방향 정보를 얻어 위치 제어와 속도 제어에 활용합니다.                                                                                            | 단순히 “도는 모터”가 아니라, 현재 얼마나 돌았는지와 얼마나 빠르게 도는지를 피드백할 수 있는 모터입니다.                                                      |
| 엔코더 부의 역할                | 엔코더는 모터 축의 회전을 전기 신호로 바꾸어 위치, 속도, 방향 정보를 제어기에 전달합니다.                                                                                                                        | 폐루프 제어에서 목표값과 실제값을 비교하는 기준 데이터를 제공합니다.                                                                            |
| 엔코더 부의 구성 방식과 작동 원리      | 증분형은 기준점으로부터의 변화량을 펄스로 계산절대형은 각 위치마다 고유한 값을 제공<br>증분형의 대표적인 쿼드러처 엔코더는 A상과 B상 두 채널의 위상 차를 이용해 위치와 방향을 판별                                                                     |                                                                                                                   |
| 펄스와 해상도의 개념 및 관계         | 펄스는 엔코더가 회전 중에 출력하는 신호의 개수이고, 해상도는 한 바퀴를 얼마나 잘게 나눠 읽을 수 있는지를 뜻합니다. 같은 바퀴 원주라면 PPR이 클수록 1펄스당 이동거리가 작아지므로 더 세밀하게 측정할 수 있습니다.                                                  |                                                                                                                   |
| 속도 계산                    | 속도는 일정 시간 동안 들어온 펄스 수를 이용해 계산할 수 있으며, 펄스 간 시간 간격을 이용해 속도 제어를 할 수 있습니다. 예를 들어 회전속도는 “일정 시간의 펄스 수 / PPR”로 회전수를 구한 뒤 시간 기준으로 환산하는 방식으로 계산합니다.                                  | $\text{회전수(rev)} = \frac{\Delta p}{\text{PPR}}$<br>$\text{속도} \propto \frac{\Delta p}{\text{PPR} \cdot \Delta t}$ |
| 거리 계산                    | 거리 계산은 바퀴 원주와 엔코더 해상도를 이용해 구합니다.                                                                                                                                            | $K_p = \frac{\text{바퀴 원주}}{\text{PPR}}$<br>$\text{총 이동거리} = \text{총 펄스 수} \times K_p$                             |
| 엔코더 데이터와 실제 주행 결과 차이의 원인 | 엔코더 기반 거리 계산은 바퀴 원주와 PPR이 정확하다는 가정에 의존하므로, 실제 유효 원주가 다르면 계산 거리와 실제 거리가 달라질 수 있습니다. 또한 A상과 B상 신호를 읽는 과정에서 처리 속도가 느리거나 노이즈가 있으면 누락 스텝이나 잘못된 판정이 발생할 수 있습니다.                   |                                                                                                                   |
| 오차 보정 방법                 | A상·B상 쿼드러처 복호를 정확히 적용하고, 노이즈나 잘못된 상태 전이를 줄이도록 신호 처리를 보강해야 합니다.  환경 조건이 거칠거나 오염 가능성이 있으면 엔코더 방식도 용도에 맞게 선택해야 하며, 일반적으로 광학식은 더 높은 해상도와 위치 정확도를 제공하고 자기식은 환경 내성이 장점이 될 수 있습니다. | 실제 주행으로 기준 거리를 여러 번 측정해 1펄스당 이동거리 상수를 다시 보정하는 방법이 기본                                                              |
