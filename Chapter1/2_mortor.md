## 수행목표
1. 핵심 부품 모터에 대한 학습

### 모터 기본 유형
| 항목    | DC 모터                                                         | 서보 모터                                          |
| ----- | ------------------------------------------------------------- | ---------------------------------------------- |
| 정의    | 직류 전원을 사용해 회전하는 기본적인 모터이며, 브러시와 정류자를 통해 전류 방향을 바꿔 연속 회전을 만든다. | 모터에 제어회로와 피드백 센서를 결합해 위치·속도·토크를 정밀하게 제어하는 구조다. |
| 장점    | 저렴하고 구조가 단순하며 속도 제어가 비교적 쉽다.                                  | 정밀 제어가 가능하고 응답성이 좋아 로봇과 자동화 장비에 적합하다.          |
| 단점    | 브러시 마모, 소음, 유지보수 부담이 있다.                                      | 가격이 높고 제어 회로가 필요하다.                            |
| 대표 용도 | 완구, 팬, 소형 구동 장치 등에 많이 쓰인다.                                    | 로봇, 드론, 자동화 장비, 정밀 기계에 널리 쓰인다.                 |
### 모터 사양 구성 항목

| 사양           | 내용                                          |
| ------------ | ------------------------------------------- |
| 정격 전압, 정격 전류 | 모터 정상 동작을 위한 설계값                            |
| 정격 회전 속도     | $min^{-1}$ 또는 RPM 형태로 표시, 모터가 정상 조건에서 도는 조건 |
| 출력           | 단위 : kW, 모터의 힘                              |
| 효율 등급        | 전력대비 얼마나 효율적으로 힘을 내주는지 표현                   |
| 기계 치수        | 프레임 번호, 축 지름, 허용 오차 등                       |
| 서보 관련 성능     | 위치, 속도, 토크 제어 성능 등 포함                       |

### 제어 원리

| 항목    | DC모터                  | 서보모터          |
| ----- | --------------------- | ------------- |
| 회전 속도 | 인가 전압 조정, PWM의 듀티비 조절 | PWM 제어 신호로 조절 |
### 모터 드라이버
 - 모터 드라이버는 아두이노나 ESP32 같은 제어 보드의 신호를 받아, 별도의 모터 전원으로 큰 전류를 안전하게 모터에 공급하는 전원 공급 장치
 - 정회전·역회전 전환, PWM 기반 속도 제어, 역기전력과 노이즈 완화, 과전류·과열 위험 감소 같은 역할도 맡는다.
 - 입문용으로는 L298N이 많이 쓰이지만 전압 강하와 발열이 큰 편이고, TB6612 계열은 상대적으로 효율이 높고 발열이 적어 소형 로봇에 더 잘 맞는 편이다.
