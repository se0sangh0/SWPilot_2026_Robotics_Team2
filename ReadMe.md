## 📌 레포지토리 소개

이 레포지토리는 **SW파일럿 2026 로보틱스 2팀**의 팀 학습 저장소입니다.  
팀원 각자가 개인 포크(Fork) 레포지토리에서 학습 내용을 정리하고,  
과정 종료 후 Pull Request를 통해 본 레포지토리에 기여합니다.

---

## 🗂️ 디렉토리 구조

```
SWPilot_2026_Robotics_Team2/
├── 팀원이름_A/          # 팀원 A의 학습 디렉토리
├── 팀원이름_B/          # 팀원 B의 학습 디렉토리
├── 팀원이름_C/          # 팀원 C의 학습 디렉토리
└── ...
```

> 각 팀원의 디렉토리 안에서 개인 학습 내용을 자유롭게 관리합니다.

---

## 🔄 참여 방법 (Fork & PR 워크플로우)

### 1단계 — Fork

본 레포지토리를 개인 GitHub 계정으로 **Fork**합니다.

```
https://github.com/se0sangh0/SWPilot_2026_Robotics_Team2
```

### 2단계 — 개인 작업

포크한 레포지토리에서 **자신의 이름으로 된 디렉토리**를 생성하고, 학습 내용을 정리합니다.

```
본인이름/
├── week01/
├── week02/
└── ...
```

### 3단계 — Pull Request 제출

과정 종료 후, 포크한 레포지토리에서 본 레포지토리로 **Pull Request**를 생성합니다.

- PR 제목 형식: `[이름] 학습 내용 제출`
- PR 본문에 학습한 내용 간략 요약 포함

### 4단계 — 머지 및 반영

관리자가 PR을 검토 후 본 레포지토리에 머지합니다.  
머지가 완료되면 팀원별 디렉토리가 메인 레포지토리에 반영됩니다.
