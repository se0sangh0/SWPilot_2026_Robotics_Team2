## 🔄 참여 방법 (Fork & PR 워크플로우)

### 1단계 — Fork

본 레포지토리를 개인 GitHub 계정으로 **Fork**합니다.

```
https://github.com/se0sangh0/SWPilot_2026_Robotics_Team2
```

### 2단계 — 개인 작업

포크한 레포지토리에서 **자신의 이름으로 된 디렉토리** 안에 챕터별로 학습 내용을 정리합니다.

```
본인이름/
└── Chapter1/
    ├── week01.md
    ├── week02.md
    └── ...
```

> ⚠️ **주차별 PR**과 **챕터**는 별개로 운영됩니다.  
> 주차 PR은 매 주차 학습 내용을 제출하는 용도이며,  
> 챕터 디렉토리는 챕터 단위로 학습 내용을 묶어서 관리하는 구조입니다.

### 3단계 — Pull Request 제출

주차 학습이 완료되면, 포크한 레포지토리에서 본 레포지토리로 **Pull Request**를 생성합니다.

- PR 제목 형식: `[이름] N주차 학습 내용 제출`
- PR 본문에 해당 주차에 학습한 내용 간략 요약 포함

### 4단계 — 머지 및 반영

관리자가 PR을 검토 후 본 레포지토리에 머지합니다.  
머지가 완료되면 팀원별 디렉토리가 메인 레포지토리에 반영됩니다.
